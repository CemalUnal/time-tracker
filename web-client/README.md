# Time Tracker

A simple time tracking application built with Spring MVC and ReactJS.



## Usage

#### Prerequisites

- Docker
- Docker compose

```bash
docker-compose up --detach
```

Open `http://localhost:8081` for web client.

## Development

#### Prerequisites

- JDK 1.8+
- Maven 3+
- Node.js 8+
- npm 5+
- MongoDB 3+
